Plugin Name: Categories Links Plugin
Plugin Uri: https://github.com/alnever/gallery-shortcode
Description: Provides a widget to insert some categories links as a block
Version: 1.0
Author: Alex Neverov
Author URI: http://alneverov.ru


The plugin allows to add a block of 3 custom links to the categories.
The widget has a configuration pane, allowing to choose catrgories, colors, and icons to represent categories in the block.
